# 载人登月任务轨道设计方法总结

## 摘要

本文档详细描述了在平面圆型限制性三体问题 (PCR3BP) 框架下，设计载人登月往返轨道的完整方法。任务包括从地球 400 km 轨道出发，经 L1 点补给站转移至月球 100 km 轨道，完成科学任务后返回地球。最终实现登月运载量 **12,197 kg**，总任务时间 **57.81 天**，满足所有约束条件。

## 2. 动力学模型

### 2.1 平面圆型限制性三体问题 (PCR3BP)

#### 2.1.1 伪势函数与运动方程

在地月会合（旋转）坐标系中，定义**伪势函数**：

$$\Omega = \frac{1}{2}(x^2 + y^2) + \frac{1-\mu}{r_1} + \frac{\mu}{r_2}$$

运动方程可由拉格朗日方程推导：

$$\ddot{x} - 2\dot{y} = \frac{\partial \Omega}{\partial x} = x - \frac{(1-\mu)(x+\mu)}{r_1^3} - \frac{\mu(x-1+\mu)}{r_2^3}$$

$$\ddot{y} + 2\dot{x} = \frac{\partial \Omega}{\partial y} = y - \frac{(1-\mu)y}{r_1^3} - \frac{\mu y}{r_2^3}$$

其中：
- $\mu = \mu_m / (\mu_e + \mu_m) = 0.01215$ 为质量参数
- $r_1 = \sqrt{(x+\mu)^2 + y^2}$ 为到地球的距离
- $r_2 = \sqrt{(x-1+\mu)^2 + y^2}$ 为到月球的距离
- 项 $-2\dot{y}$ 和 $2\dot{x}$ 为**科里奥利加速度**，源于旋转坐标系

#### 2.1.2 状态空间表示

将二阶 ODE 转化为一阶状态方程，状态向量 $\mathbf{X} = [x, y, v_x, v_y]^T$：

$$\dot{\mathbf{X}} = \begin{bmatrix} v_x \\ v_y \\ 2v_y + \partial\Omega/\partial x \\ -2v_x + \partial\Omega/\partial y \end{bmatrix}$$

#### 2.1.3 Jacobi 积分（能量守恒）

CR3BP 存在唯一的运动积分——**Jacobi 常数**：

$$C_J = 2\Omega - (v_x^2 + v_y^2) = 2\Omega - v^2$$

在无机动的自由飞行中，$C_J$ 严格守恒，可用于：
1. **数值精度检验**：积分过程中 $C_J$ 变化应 < 10⁻¹⁰
2. **可达性分析**：Jacobi 常数确定了零速度曲线，划定可达区域边界

### 2.2 单位系统

采用无量纲化处理简化计算并提高数值稳定性：

| 量纲 | 归一化单位 | 物理值 | 定义 |
|------|-----------|--------|------|
| 长度 [LU] | 1 | 384,400 km | 地月平均距离 |
| 时间 [TU] | 1 | 375,190 s ≈ 4.34 天 | 使得角速度 ω = 1 |
| 速度 [VU] | 1 | 1.0245 km/s | LU/TU |
| 质量 [MU] | 1 | $M_e + M_m$ | 系统总质量 |

**单位转换关系**：
- 时间：$t_{TU} = t_{day} / 4.342$
- 速度：$v_{VU} = v_{km/s} / 1.0245$
- 加速度：$a_{normalized} = a_{km/s^2} \times TU^2/LU$

### 2.3 旋转坐标系中的圆轨道速度

在会合坐标系中，绕天体的圆轨道速度需要修正。设航天器相对中心天体的位置为 $\mathbf{r}_{rel}$，则：

**开普勒圆轨道速度**（惯性系）：
$$v_{circ} = \sqrt{\mu_{body} / r}$$

**旋转坐标系修正**：
$$\mathbf{v}_{syn} = v_{circ} \cdot \hat{\theta} - \boldsymbol{\omega} \times \mathbf{r}_{rel}$$

其中：
- $\boldsymbol{\omega} = [0, 0, 1]^T$ 为归一化角速度向量
- $\hat{\theta} = [-r_y/r, r_x/r]^T$ 为轨道切向单位向量（逆时针为正）
- $\boldsymbol{\omega} \times \mathbf{r}_{rel} = [-r_y, r_x]^T$ 为坐标系旋转带来的速度

**物理解释**：在旋转坐标系中观测，圆轨道上的航天器需要"抵消"坐标系的旋转效应。

### 2.4 状态转移矩阵 (STM)

状态转移矩阵描述微小偏差的演化，是流形计算和轨道修正的核心工具。

#### 2.4.1 定义与计算

对于状态偏差 $\delta\mathbf{X}$，其演化满足线性化方程：
$$\delta\dot{\mathbf{X}} = \mathbf{A}(t) \cdot \delta\mathbf{X}$$

其中 $\mathbf{A}(t)$ 为 Jacobi 矩阵：

$$\mathbf{A} = \begin{bmatrix}
0 & 0 & 1 & 0 \\
0 & 0 & 0 & 1 \\
\Omega_{xx} & \Omega_{xy} & 0 & 2 \\
\Omega_{xy} & \Omega_{yy} & -2 & 0
\end{bmatrix}$$

**伪势函数的二阶偏导数**：
$$\Omega_{xx} = 1 - \frac{1-\mu}{r_1^3} - \frac{\mu}{r_2^3} + \frac{3(1-\mu)(x+\mu)^2}{r_1^5} + \frac{3\mu(x-1+\mu)^2}{r_2^5}$$

$$\Omega_{yy} = 1 - \frac{1-\mu}{r_1^3} - \frac{\mu}{r_2^3} + \frac{3(1-\mu)y^2}{r_1^5} + \frac{3\mu y^2}{r_2^5}$$

$$\Omega_{xy} = \frac{3(1-\mu)(x+\mu)y}{r_1^5} + \frac{3\mu(x-1+\mu)y}{r_2^5}$$

#### 2.4.2 数值积分

STM 与状态向量同时积分，扩展状态向量为 20 维：
$$\mathbf{Y} = [\mathbf{X}^T, \Phi_{11}, \Phi_{21}, ..., \Phi_{44}]^T$$

STM 演化方程：
$$\dot{\Phi}(t) = \mathbf{A}(t) \cdot \Phi(t), \quad \Phi(0) = \mathbf{I}_{4\times4}$$

#### 2.4.3 应用

1. **偏差传播**：$\delta\mathbf{X}(t) = \Phi(t,t_0) \cdot \delta\mathbf{X}(t_0)$
2. **流形方向计算**：Monodromy 矩阵的特征向量
3. **打靶法修正**：$\Delta\mathbf{X}_0 = \Phi^{-1} \cdot \Delta\mathbf{X}_{target}$

## 3. L1 Lyapunov 轨道计算

### 3.1 问题描述

补给飞船位于 L1 点附近的 Lyapunov 周期轨道上，初始条件为：
- 位置：$\mathbf{r}(0) = [0.8, 0]^T$ LU
- 需求解：初始速度 $\mathbf{v}(0)$ 和轨道周期 $T$

### 3.2 L1 平动点计算

L1 点是共线平动点，满足方程：
$$x - \frac{(1-\mu)(x+\mu)}{|x+\mu|^3} - \frac{\mu(x-1+\mu)}{|x-1+\mu|^3} = 0, \quad y = 0$$

使用 **牛顿-拉夫森迭代法** 求解：

```matlab
f(x) = x - (1-μ)(x+μ)/|x+μ|³ - μ(x-1+μ)/|x-1+μ|³
f'(x) = 1 + 2(1-μ)/|x+μ|³ + 2μ/|x-1+μ|³

迭代: x_{n+1} = x_n - f(x_n)/f'(x_n)
初值: x_0 = 0.8 (介于地球和月球之间)
收敛条件: |f(x)| < 10⁻¹⁴
```

结果：$x_{L1} = 0.8369$ LU

### 3.3 Lyapunov 轨道求解方法

#### 3.3.1 对称性约束

利用 Lyapunov 轨道的 **x 轴对称性**：
- 初始状态：$\mathbf{X}_0 = [x_0, 0, 0, v_{y0}]^T$（在 x 轴上，速度垂直于 x 轴）
- 半周期条件：$y(T/2) = 0$ 且 $v_x(T/2) = 0$

#### 3.3.2 差分修正算法

**Step 1**：初始猜测（线性化动力学）

在 L1 附近线性化，特征方程：
$$\lambda^4 + (2 - \Omega_{xx})\lambda^2 + ... = 0$$

对于 L1 点，存在一对纯虚特征值 $\pm i\omega$，对应周期运动。近似关系：
$$v_{y0} \approx -s \cdot A_x, \quad s = \sqrt{\Omega_{xx}(x_{L1}) - 1}$$

其中 $A_x = |x_0 - x_{L1}|$ 为 x 方向振幅。

**Step 2**：单变量求根

利用对称性，只需调整 $v_{y0}$ 使得 $v_x(T/2) = 0$。使用 **fzero** 求解：

```matlab
目标函数: f(v_y0) = v_x(t*) where y(t*) = 0
求根算法: Brent 方法 (fzero)
精度设置: TolX = 10⁻¹⁴
```

**Step 3**：事件检测

使用 ODE 事件函数精确检测 $y = 0$ 的穿越时刻：

```matlab
function [value, isterminal, direction] = y_cross(~, X, dir)
    value = X(2);           % y = 0
    isterminal = 1;         % 停止积分
    direction = dir;        % +1: 从下往上; -1: 从上往下
end
```

### 3.4 数值实现细节

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始 x | 0.8 LU | 指定的 Lyapunov 轨道振幅 |
| fzero TolX | 10⁻¹⁴ | 根求解精度 |
| ODE RelTol | 10⁻¹² | 相对误差容差 |
| ODE AbsTol | 10⁻¹⁴ | 绝对误差容差 |
| MaxStep | 0.02 TU | 最大积分步长 |

### 3.5 求解结果

| 参数 | 值 |
|------|-----|
| 初始速度 $v_{y0}$ | 0.3556 VU |
| 轨道周期 $T$ | 3.32 TU (14.4 天) |
| x 方向振幅 | 0.037 LU (14,200 km) |

### 3.6 验证

1. **周期性验证**：积分一个完整周期后，状态误差 < 10⁻¹² LU
2. **Jacobi 常数守恒**：全程 $\Delta C_J < 10⁻¹¹$
3. **对称性检验**：$|y(T/2)|$ 和 $|v_x(T/2)|$ < 10⁻¹³

## 4. Phase 1: LEO → L1 转移设计

### 4.1 设计策略

采用**负 C₃ 策略**：利用较低的出发速度换取更大的发射质量。

**发射质量模型**（题目给定）：
$$M_0 = 25000 - 1000 \cdot C_3 \quad [\text{kg}]$$

**C₃ 计算**（相对地球的二体特征能量）：

在惯性系中，航天器相对地球的状态：
$$C_3 = v_{rel}^2 - \frac{2\mu_e}{r_{rel}} \quad [\text{km}^2/\text{s}^2]$$

其中 $r_{rel}$ 和 $v_{rel}$ 为相对地球的位置和速度（需从旋转坐标系转换到惯性系后计算）。

**策略优势**：负 C₃ 意味着航天器能量低于逃逸能量，需要更长的飞行时间，但可显著增加发射质量。

### 4.2 坐标变换

#### 4.2.1 旋转系 → 惯性系

设旋转坐标系状态为 $(x, y, v_x, v_y)$，时刻 $t$（归一化），则惯性系状态：

**位置变换**：
$$\begin{bmatrix} x_I \\ y_I \end{bmatrix} = \begin{bmatrix} \cos t & -\sin t \\ \sin t & \cos t \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix}$$

**速度变换**（包含旋转效应 $\omega \times r$）：
$$\begin{bmatrix} v_{x,I} \\ v_{y,I} \end{bmatrix} = \begin{bmatrix} \cos t & -\sin t \\ \sin t & \cos t \end{bmatrix} \begin{bmatrix} v_x - y \\ v_y + x \end{bmatrix}$$

#### 4.2.2 相对地球状态

地球在惯性系中的位置和速度：
$$\mathbf{r}_{Earth} = [-\mu\cos t, -\mu\sin t]^T, \quad \mathbf{v}_{Earth} = [\mu\sin t, -\mu\cos t]^T$$

相对状态：
$$\mathbf{r}_{rel} = \mathbf{r}_I - \mathbf{r}_{Earth}, \quad \mathbf{v}_{rel} = \mathbf{v}_I - \mathbf{v}_{Earth}$$

### 4.3 优化问题数学描述

**设计变量** $\mathbf{x} = [\theta, t_{dep}, ToF, \Delta v_x, \Delta v_y]^T$：
- $\theta$: LEO 出发相位角 [rad]
- $t_{dep}$: 出发时刻 [TU]
- $ToF$: 飞行时间 [TU]
- $(\Delta v_x, \Delta v_y)$: 出发速度增量分量 [VU]

**目标函数**：
$$\min J = C_3 + 50 \cdot \|\Delta\mathbf{v}_{match}\|^2$$

其中 $\Delta\mathbf{v}_{match}$ 为 L1 对接所需速度匹配。权重因子 50 平衡两项的量级。

**等式约束**（位置匹配）：
$$h(\mathbf{x}) = \begin{bmatrix} x_{arrival} - x_{supply}(t_{arr}) \\ y_{arrival} - y_{supply}(t_{arr}) \end{bmatrix} = \mathbf{0}$$

其中 $t_{arr} = t_{dep} + ToF$，供给船位置通过在 Lyapunov 轨道上插值获得。

**不等式约束**（路径约束，通过 ODE 事件检测）：
1. $r_{Earth} \geq R_e + 400$ km（近地点高度）
2. $r_{Moon} \geq R_m + 100$ km（近月点高度）
3. $r_{barycenter} \leq 2$ LU（不逃逸）

### 4.4 数值求解方法

#### 4.4.1 两阶段优化策略

**阶段 1：网格搜索（全局探索）**

```
搜索空间离散化:
  θ:        36 个点 × [0, 2π]
  t_dep:    12 个点 × [0, T_lyap]
  ToF:      15 个点 × [1.5, 6] TU
  burn_ang:  5 个点 × [-15°, +15°]

总计: 32,400 个候选点
```

对每个候选点：
1. 计算 LEO 初始状态 $\mathbf{X}_0(\theta)$
2. 施加 $\Delta\mathbf{v}$ 后积分 $ToF$ 时间
3. 评估目标函数和约束违反程度
4. 保留 top-N 最优候选

**阶段 2：局部优化（fmincon）**

以网格搜索最优解为初值，使用 **SQP 算法**（序列二次规划）：

```matlab
options = optimoptions('fmincon', ...
    'Algorithm', 'sqp', ...
    'MaxIterations', 1000, ...
    'OptimalityTolerance', 1e-10, ...
    'ConstraintTolerance', 1e-9, ...
    'FiniteDifferenceType', 'central', ...
    'FiniteDifferenceStepSize', 1e-8);
```

**SQP 算法原理**：
1. 在当前点构造二次规划子问题（目标函数二次近似 + 约束线性化）
2. 求解 QP 得到搜索方向
3. 线搜索确定步长
4. 迭代直至满足 KKT 条件

#### 4.4.2 梯度计算

使用 **中心差分法** 估计梯度（显式解析梯度复杂度高）：

$$\frac{\partial J}{\partial x_i} \approx \frac{J(x_i + h) - J(x_i - h)}{2h}, \quad h = 10^{-8}$$

中心差分精度为 $O(h^2)$，优于前向差分的 $O(h)$。

#### 4.4.3 精度提升（可选的第三阶段）

若 fmincon 结果位置误差 > 10⁻⁸ LU，执行二次精化：

```matlab
更严格的容差设置:
  OptimalityTolerance = 1e-12
  ConstraintTolerance = 1e-10
  FiniteDifferenceStepSize = 1e-10
```

### 4.5 ODE 事件检测实现

**路径约束事件函数**：

```matlab
function [value, isterminal, direction] = path_events(~, X, p)
    r_earth = sqrt((X(1)+p.mu)^2 + X(2)^2);
    r_moon  = sqrt((X(1)-1+p.mu)^2 + X(2)^2);
    r_bary  = sqrt(X(1)^2 + X(2)^2);

    value = [r_earth - p.r_LEO;    % 不低于 LEO

end
```

## 5. Phase 2: L1 → LLO 转移设计

### 5.1 设计策略

利用 **Lyapunov 轨道的不稳定流形**自然漂移到月球附近，减少燃料消耗。

**核心思想**：Lyapunov 轨道是不稳定的，其附近存在不稳定流形（远离轨道）和稳定流形（趋近轨道）。适当的微小扰动可使航天器沿不稳定流形自然滑向月球，几乎不消耗燃料。

### 5.2 流形理论基础

#### 5.2.1 Monodromy 矩阵

将 Lyapunov 轨道上的初始点积分一个完整周期 $T$，得到 **单值矩阵**（Monodromy Matrix）：
$$\mathbf{M} = \Phi(T, 0)$$

$\mathbf{M}$ 描述了沿轨道经过一个周期后微小偏差的演化。

#### 5.2.2 特征值分析

$\mathbf{M}$ 的特征值反映轨道的稳定性：

| 特征值类型 | 条件 | 数量 | 物理意义 |
|-----------|------|------|----------|
| 不稳定 | $|\lambda| > 1$ | 1 | 偏差指数增长 |
| 稳定 | $|\lambda| < 1$ | 1 | 偏差指数衰减 |
| 中性 | $|\lambda| \approx 1$ | 2 | 沿轨道和能量方向 |

**特征值分类算法**：
```matlab
[V, D] = eig(M);
lambda = diag(D);

tol_neutral = 1e-3;  % 中性判定容差

for i = 1:4
    if abs(abs(lambda(i)) - 1) < tol_neutral
        % 中性特征值
    elseif abs(lambda(i)) > 1
        % 不稳定特征值 → 对应不稳定流形
        v_unstable = V(:, i);
    else
        % 稳定特征值 → 对应稳定流形
        v_stable = V(:, i);
    end
end
```

#### 5.2.3 特征向量沿轨道传播

轨道上任意点的特征向量可由初始点传播得到：
$$\mathbf{v}_u(t) = \frac{\Phi(t,0) \cdot \mathbf{v}_{u,0}}{\|\Phi(t,0) \cdot \mathbf{v}_{u,0}\|}$$

**符号连续性修正**：由于特征向量方向不唯一（$\pm$），需确保沿轨道连续：
```matlab
if dot(v_new, v_prev) < 0
    v_new = -v_new;
end
```

### 5.3 流形生成算法

#### 5.3.1 算法流程

```
输入: Lyapunov 轨道状态 {X_i, t_i}, 扰动量 ε
输出: 流形轨迹库

1. 计算参考点的 Monodromy 矩阵 M = Φ(T, 0)
2. 特征分解: [V, D] = eig(M), 识别 v_u, v_s
3. FOR each 轨道点 i:
   a. 计算该点的特征向量: v_u(i) = Φ(t_i) · v_u(0) / ||...||
   b. 提取速度分量: v_u_vel = v_u(i)[3:4] / ||v_u(i)[3:4]||
   c. 施加扰动: X_pert = X_i + [0; 0; ε·v_u_vel]
   d. 正时间积分至 t_max 或触发事件
   e. 记录到达 LLO 高度的轨迹
4. END FOR
```

#### 5.3.2 扰动方式

**关键设计决策**：扰动仅施加在**速度空间**，而非位置-速度全状态。

$$\Delta\mathbf{X}_{pert} = [0, 0, \varepsilon \cdot \hat{v}_{u,x}, \varepsilon \cdot \hat{v}_{u,y}]^T$$

**原因**：
1. 速度扰动可由推进器精确实现（即流形注入机动）
2. 位置扰动物理上难以实现
3. 纯速度扰动对应实际的轨道机动

#### 5.3.3 扰动量级选择

**关键参数**：$\varepsilon = 10^{-3}$ VU $\approx 1.0$ m/s

| 扰动量级 | 物理值 | 问题 |
|---------|--------|------|
| 10⁻⁶ VU | ~0.001 m/s | 与积分误差同量级，不可靠 |
| 10⁻⁴ VU | ~0.1 m/s | 边缘情况，流形特性可能不稳定 |
| **10⁻³ VU** | **~1.0 m/s** | 物理可靠，燃料消耗可接受 |
| 10⁻² VU | ~10 m/s | 燃料浪费，偏离流形较远 |

### 5.4 ODE 事件检测

#### 5.4.1 流形终止条件

积分在以下任一条件满足时停止：

```matlab
function [value, isterminal, direction] = manifold_events(~, X, p)
    r_moon = sqrt((X(1)-1+p.mu)^2 + X(2)^2);  % 距月球
    r_earth = sqrt((X(1)+p.mu)^2 + X(2)^2);   % 距地球
    r_bary = sqrt(X(1)^2 + X(2)^2);           % 距重心

    value = [r_moon - p.r_LLO;     % 到达 LLO 高度
             r_earth - p.r_LEO;    % 到达 LEO 高度（稳定流形）
             r_bary - 2.0];        % 逃逸边界

    isterminal = [1; 1; 1];
    direction = [0; 0; 0];         % 双向检测
end
```

其中：
- $r_{LLO} = (R_m + 100 \text{ km}) / LU = 0.00478$ LU
- $r_{LEO} = (R_e + 400 \text{ km}) / LU = 0.01764$ LU

#### 5.4.2 事件精度

MATLAB ODE 事件检测基于**零点插值**：当 `value` 变号时，使用三次 Hermite 插值精确定位零点，精度可达积分步长的 10⁻⁶ 倍。

### 5.5 候选轨道筛选与选择

#### 5.5.1 筛选标准

对于生成的流形轨迹库，按以下条件筛选：

| 条件 | 阈值 | 原因 |
|------|------|------|
| LLO 入轨 Δv | < 1.5 km/s | 燃料限制 |
| 总飞行时间 | < 80 天 | 预留返回时间 |
| 入射角 | < 90° | 避免垂直撞击 |

#### 5.5.2 入轨 Δv 计算

在旋转坐标系中，月球环绕轨道速度：

**相对月球的位置和距离**：
$$\mathbf{r}_{rel} = [x - (1-\mu), y]^T, \quad r = \|\mathbf{r}_{rel}\|$$

**开普勒圆轨道速度**（惯性系）：
$$v_{circ} = \sqrt{\mu_m / r}$$

**旋转坐标系中的目标速度**：
$$\mathbf{v}_{target} = v_{circ} \cdot \hat{\theta} - \boldsymbol{\omega} \times \mathbf{r}_{rel}$$

其中 $\hat{\theta} = [-r_y/r, r_x/r]^T$（顺行）或 $[r_y/r, -r_x/r]^T$（逆行）

**入轨 Δv**：
$$\Delta v_{insert} = \|\mathbf{v}_{target} - \mathbf{v}_{arrival}\|$$

选择顺行/逆行中 Δv 较小的方案。

#### 5.5.3 最优候选选择

在满足筛选条件的候选中，选择使登月运载量最大的方案：

$$M_{carry} = M_{arrive,LLO} - M_{dry} - M_{fuel,return}$$

其中燃料消耗由齐奥尔科夫斯基方程计算。

### 5.6 流形注入机动

从 Lyapunov 轨道进入不稳定流形需要施加速度增量：

$$\Delta \mathbf{v}_{seed} = \varepsilon \cdot \hat{\mathbf{v}}_u$$

**物理实现**：
1. 航天器在 L1 补给站完成燃料加注
2. 等待至最优离开时刻（相位对齐）
3. 沿不稳定特征向量方向施加 ~1 m/s 脉冲
4. 自由滑行至月球

**燃料消耗**（由齐奥尔科夫斯基方程）：
$$\Delta m = m_0 \cdot (1 - e^{-\Delta v / v_e}) \approx 12 \text{ kg}$$

## 6. Phase 3: LLO → Earth 返回设计

### 6.1 设计目标

找到一条轨道，使得：
1. **近地点高度 = 0 km**（地球表面再入）
2. **飞船恰好位于近地点**（径向速度 = 0，确保正确再入角）

### 6.2 设计变量

| 变量 | 含义 | 搜索范围 | 说明 |
|------|------|----------|------|
| $\theta_{LLO}$ | 出发相位角 | $[0, 2\pi]$ | LLO 上的出发位置 |
| $t_{stay}$ | 月面停留时间 | 3 ~ 10 天 | 任务约束 |
| $|\Delta v|$ | 速度增量大小 | 0.8 ~ 1.5 km/s | 经验范围 |
| $\phi_{dv}$ | Δv 方向角 | $[-\pi, \pi]$ | 相对速度方向的偏转 |

**Δv 方向参数化**：
$$\Delta\mathbf{v} = |\Delta v| \cdot [\cos(\psi + \phi_{dv}), \sin(\psi + \phi_{dv})]^T$$

其中 $\psi = \arctan(v_y / v_x)$ 为当前速度方向。

### 6.3 近地点检测数学原理

#### 6.3.1 近地点定义

近地点是航天器距地球最近的点，满足：
$$\frac{dr_{Earth}}{dt} = 0 \quad \text{且} \quad \frac{d^2r_{Earth}}{dt^2} > 0$$

#### 6.3.2 径向速度计算

相对地球的径向速度：
$$\dot{r}_{Earth} = \frac{\mathbf{r}_{rel} \cdot \mathbf{v}_{rel}}{r_{Earth}} = \frac{(x+\mu)v_x + y \cdot v_y}{\sqrt{(x+\mu)^2 + y^2}}$$

**注意**：这里直接使用旋转坐标系变量，因为径向速度的符号在两个坐标系中相同。

#### 6.3.3 ODE 事件函数

```matlab
function [value, isterminal, direction] = event_periapsis(~, X, p)
    dx = X(1) + p.mu;      % 相对地球的 x 分量
    dy = X(2);             % 相对地球的 y 分量
    r = sqrt(dx^2 + dy^2); % 距地球距离

    % 径向速度 = d(r)/dt
    rdot = (dx * X(3) + dy * X(4)) / r;

    value = rdot;          % 检测零点
    isterminal = 1;        % 触发时停止
    direction = +1;        % 从负变正 = 近地点（接近→远离）
end
```

**direction = +1 的含义**：
- 接近地球时：$\dot{r} < 0$
- 经过近地点时：$\dot{r}$ 由负变正
- 远离地球时：$\dot{r} > 0$

### 6.4 优化问题数学描述

**目标函数**：
$$\min J = |r_{peri} - R_e|$$

其中 $r_{peri}$ 为近地点距地心距离，$R_e = 6378$ km 为地球半径。

**等式约束**（隐含在积分终止条件中）：
$$\dot{r}_{Earth}(t_{peri}) = 0$$

**不等式约束**：
1. $3 \leq t_{stay} \leq 10$ 天（月面停留时间）
2. $r_{Moon}(t) \geq R_m + 100$ km（全程不撞月）
3. $r_{bary}(t) \leq 2$ LU（不逃逸）
4. $t_{total} \leq 100$ 天（总任务时间）

### 6.5 三阶段求解策略

#### 6.5.1 阶段 1：粗网格搜索

## 7. 燃料规划

### 7.1 齐奥尔科夫斯基方程

火箭方程描述速度增量与质量变化的关系：

$$\Delta v = v_e \cdot \ln\frac{m_i}{m_f}$$

等价形式：
$$m_f = m_i \cdot e^{-\Delta v / v_e}$$

**燃料消耗**：
$$\Delta m_{fuel} = m_i - m_f = m_i \cdot (1 - e^{-\Delta v / v_e})$$

其中 $v_e = I_{sp} \cdot g_0 = 3000$ m/s 为排气速度（对应比冲约 306 s）。

### 7.2 质量分解

航天器总质量由以下部分组成：

$$m_{total} = m_{dry} + m_{fuel} + m_{carry}$$

| 组成 | 符号 | 值/说明 |
|------|------|---------|
| 干质量 | $m_{dry}$ | 10,000 kg（结构+设备） |
| 燃料 | $m_{fuel}$ | 最大 15,000 kg |
| 运载量 | $m_{carry}$ | 待优化目标 |

### 7.3 反向燃料规划方法

由于约束在任务末端（最终燃料 ≤ 100 kg），采用**反向规划**：从终点反推各阶段所需燃料。

#### 7.3.1 反向计算公式

设某阶段机动 $\Delta v$，**机动后**质量为 $m_{after}$，则**机动前**所需质量：

$$m_{before} = m_{after} \cdot e^{\Delta v / v_e}$$

**燃料消耗**：
$$\Delta m_{fuel} = m_{before} - m_{after} = m_{after} \cdot (e^{\Delta v / v_e} - 1)$$

#### 7.3.2 完整链条推导

**Step 4（返回地球）**：
```
目标: m_final = m_dry + m_fuel_final = 10000 + 44 = 10044 kg
Δv_return = 1.180 km/s

m_before_return = 10044 × exp(1180/3000) = 10044 × 1.482 = 14,882 kg
燃料消耗 = 14882 - 10044 = 4,838 kg
```

**Step 3（LLO 入轨）**：
```
m_after_insert = 14,882 kg (即返回前质量，因运载量在 LLO 卸载)
Δv_insert = 0.687 km/s

m_before_insert = 14882 × exp(687/3000) = 14882 × 1.257 = 18,712 kg

但需加上运载量:
m_arrive_LLO = 18712 + m_carry
```

**Step 2（流形注入）**：
```
Δv_seed = 0.001 km/s ≈ 1 m/s
质量变化极小: Δm ≈ m × (1/3000) ≈ 6 kg
```

**Step 1（L1 对接）**：
```
Δv_dock = 0.61 km/s
m_arrive_L1 × exp(-610/3000) = m_L1_docked
```

**L1 加油**：
```
m_L1_docked + m_refuel = m_depart_L1
约束: m_refuel ≤ m_fuel_capacity - m_fuel_remain
```

### 7.4 运载量优化

目标是最大化 $m_{carry}$，受以下约束：

1. **发射质量约束**：$m_0 = 25000 - 1000 \cdot C_3$
2. **燃料容量约束**：$m_{fuel} \leq 15000$ kg
3. **最终燃料约束**：$m_{fuel,final} \leq 100$ kg
4. **加油约束**：L1 可无限加油，但燃料箱有上限

**迭代求解**：
```matlab
for m_carry in search_range:
```

- CR3BP 为非刚性系统
- 长期积分（数十天）需要高阶方法减少累积误差
- 自适应步长对事件检测友好
- 多步法在光滑轨道上效率更高

#### 10.3.2 积分精度设置

```matlab
opts = odeset('RelTol', 1e-12, ...    % 相对误差
              'AbsTol', 1e-14, ...    % 绝对误差
              'MaxStep', 0.01, ...    % 最大步长 (TU)
              'Events', @event_func); % 事件检测
```

**精度选择依据**：
- 位置精度要求：10⁻⁶ LU ≈ 380 m
- 长期积分（~15 TU）误差累积因子：~10³
- 因此单步误差应 < 10⁻⁹，设置 RelTol = 10⁻¹² 留有余量

#### 10.3.3 事件检测原理

MATLAB ODE 事件检测通过以下步骤实现：

1. **符号监测**：每步检查 `value` 函数是否变号
2. **零点定位**：变号时使用三次 Hermite 插值精确定位
3. **方向判断**：`direction` 参数筛选特定穿越方向
4. **终止控制**：`isterminal = 1` 时停止积分

**精度**：零点定位精度约为步长的 10⁻⁶ 倍

#### 10.3.4 多级优化策略

```
Level 1: 网格搜索（全局探索）
    ├─ 粗糙但覆盖全面
    ├─ 可并行化 (parfor)
    └─ 识别可行区域

Level 2: fmincon SQP（局部优化）
    ├─ 从网格最优点出发
    ├─ 处理等式/不等式约束
    └─ 收敛到局部最优

Level 3: fsolve 牛顿法（精确求根）
    ├─ 仅当需要更高精度时
    ├─ 将优化问题转化为求根
    └─ 二次收敛速度
```

#### 10.3.5 数值稳定性技巧

1. **归一化单位**：避免数值量级差异过大
2. **中心差分**：梯度估计精度 O(h²) 优于前向差分 O(h)
3. **条件数监控**：STM 特征值跨越多个数量级，需注意矩阵病态
4. **Jacobi 常数检验**：每次积分后验证能量守恒

#### 10.3.6 流形扰动量级选择

扰动量 $\varepsilon$ 的选择需要平衡：

| 过小 | 过大 |
|------|------|
| 被数值误差淹没 | 偏离真实流形 |
| 流形不可靠 | 浪费燃料 |

**最优选择**：$\varepsilon \sim 10^{-3}$ VU
- 远大于积分误差（~10⁻¹⁰）
- 物理上仅需 ~1 m/s 机动
- 流形特性稳定可靠

## 12. 结论

本设计采用 **L1 中转站 + 不稳定流形转移** 策略，成功实现了载人登月往返任务：

1. **运载能力**：登月运载量 **12,197 kg**，充分利用了负 C₃ 策略
2. **燃料效率**：利用流形自然漂移，总消耗 Δv 仅 **2.48 km/s**（不含出发）
3. **任务时间**：总任务时间 **57.81 天**，远小于 100 天限制
4. **精度保证**：所有拼接点精度优于 10⁻⁶ 要求多个数量级
5. **约束满足**：全部任务约束均满足

### 12.1 设计亮点

1. **流形利用**：通过不稳定流形实现低能量月球转移，仅需 ~1 m/s 注入机动
2. **多级优化**：网格搜索 → SQP 优化 → 牛顿精修的三级策略确保全局最优
3. **高精度积分**：Adams 方法 + 严格误差控制保证长期积分精度
4. **事件检测**：精确捕捉轨道边界和近地点，实现亚米级精度

### 12.2 方法论价值

该设计展示了三体动力学在深空探测中的应用价值：
- **动力学模型**：PCR3BP 捕捉了地月系统的本质特征
- **不变流形**：自然动力学结构提供低能量转移通道
- **L1 补给站**：使得"航班化"登月成为可能
- **数值方法**：现代优化算法与高精度积分的结合
